## How to view or convert the DOCX
- To open the Word document on Windows, double-click `Afficionado_Coffee_Roasters_Research_Paper.docx` or open it in Microsoft Word / LibreOffice.
- To convert to PDF using Word: File → Save As → choose `PDF`.
- To convert from the command line (requires `libreoffice`):

```powershell
libreoffice --headless --convert-to pdf "Afficionado_Coffee_Roasters_Research_Paper.docx" --outdir .
```
