# Dashboards

This folder contains dashboard assets used for the Nassau Candy Distributor analysis project.

Files
-----
- `Sales Trend and Time-Based Performance Analysis jpg` — main dashboard image (store & product insights).
- `Readme.md` — this file.

About the dashboard
-------------------
The dashboard visualizes sales and transaction patterns across stores and product categories. Key sections include:

- KPI tiles: maximum revenue, total transactions, number of products, top store location.
- Revenue by product detail and product type (bar charts).
- Transaction quantity by time bucket (donut chart) and by hour (line chart).
- Revenue breakdown by product category (pie chart) and by store location (bar chart).

How to view
-----------
- Open the JPG in an image viewer or in VS Code to inspect visuals.
- For an interactive version, open the original Power BI / Tableau workbook if available.

Notes for interpretation
------------------------
- Use the time-bucket and store-location filters to focus on subsets of the data.
- Pivot tables and calculated columns (e.g., Lead_Time, Delayed_Flag) were used to generate insights.
